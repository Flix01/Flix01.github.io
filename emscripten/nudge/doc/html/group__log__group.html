User can define NUDGE_VLOG_FUNC(CONST_CHAR_PTR_ARG_PTR , VA_LIST_ARG) (default: vfprintf(NUDGE_LOG_FILE_PTR,CONST_CHAR_PTR_ARG_PTR,VA_LIST_ARG)) to have better control
